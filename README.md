# homebrew-tap

Homebrew tap for fanbuz's CLI tools.

## Available Formulae

- `gitea-cli`

## Usage

```bash
brew tap fanbuz/tap
brew install fanbuz/tap/gitea-cli
```

## Upgrade

```bash
brew update
brew upgrade gitea-cli
```

## Development

Formula files live under `Formula/`.

Current structure:

```text
Formula/
  gitea-cli.rb
```
